## 🚀 Quick Start (3 Steps)

### **Step 1 — Extract Files**

Unzip the project folder to:

```
E:\SNS IT\project\Medical
```

### **Step 2 — Run the Application**

**Windows**
Double-click:

```
RUN.bat
```

**Mac / Linux**
Open Terminal in the project folder and run:

```bash
./RUN.sh
```

### **Step 3 — Open in Browser**

Go to:

```
http://localhost:5000
```

The website should open automatically.

🎉 **Done!**

## 🛠️ Troubleshooting

### ❌ RUN.bat doesn’t work (Windows)

* Check Python installation:

  ```bash
  python --version
  ```
* Try manual start:

  ```bash
  python app.py
  ```

### ❌ RUN.sh doesn’t work (Mac/Linux)

```bash
cd /path/to/MedicalImageAnalyzer
chmod +x RUN.sh
./RUN.sh
```

### ❌ Website won’t load

* Wait 5–10 seconds after starting server
* Check terminal for server status
* Open manually:

  ```
  http://localhost:5000
  ```

### ❌ Dependencies failed to install

```bash
pip install -r requirements.txt
```
