================================================================================
                        TROUBLESHOOTING
================================================================================

❌ RUN.bat doesn't work?
   → Make sure Python is installed: python --version
   → Try running Command Prompt and type: python app.py

❌ RUN.sh doesn't work?
   → Open Terminal
   → Navigate to folder: cd /path/to/MedicalImageAnalyzer
   → Run: chmod +x RUN.sh
   → Then: ./RUN.sh

❌ Website won't load?
   → Wait 5-10 seconds after running
   → Check if server is running in terminal
   → Try: http://localhost:5000

❌ Dependencies not installing?
   → Open terminal in folder
   → Run: pip install -r requirements.txt
